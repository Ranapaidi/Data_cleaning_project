# 📊 Data Cleaning & Reporting Automation

## 🚀 Overview

This project cleans and analyzes an employee salary dataset using Python.

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## ✨ Features

* Handle missing values
* Remove duplicate data
* Generate summary reports
* Create data visualizations
* Export cleaned dataset

## 📂 Dataset

Employee Salary Dataset (CSV)
